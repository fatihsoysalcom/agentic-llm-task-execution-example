# Agentic LLM Task Execution Example

This example demonstrates a simplified "agentic" behavior for a large language model (LLM). It simulates an agent that breaks down a complex goal into sequential steps, uses an LLM (mocked here) to perform each step, and progresses towards the goal, illustrating goal-oriented, multi-step task execution.

## Language

`python`

## How to Run

Save the code as `agent_demo.py`. Run it from your terminal using `python agent_demo.py`.
